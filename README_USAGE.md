## 🔐 登录方式

**手机号 + 验证码登录**

1. 打开 http://localhost:5001
2. 输入手机号（11 位，不含 +86）
3. 点击"获取验证码"
4. 查看系统日志获取验证码（测试环境会显示在控制台）
5. 输入验证码登录

**测试验证码查看方法：**
```bash
tail -f /tmp/trust-system.log
```
验证码会显示在日志中，格式：`验证码已生成 - 手机号：15382303557, 验证码：123456`
