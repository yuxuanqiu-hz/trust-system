## 🔒 安全说明

### 当前配置

- ✅ HTTPS 加密传输（Cloudflare Tunnel）
- ✅ 密码加密存储（Werkzeug）
- ✅ 会话管理（Flask-Login）
- ✅ 权限控制（管理员/员工）

### 建议的安全措施

1. **强制修改初始密码**
   - 员工首次登录后必须修改密码
   - 密码长度至少 8 位
   - 包含大小写字母 + 数字

2. **定期修改密码**
   - 建议每 90 天修改一次
   - 不要使用相同密码

3. **账号安全**
   - 离职员工及时禁用账号
   - 定期检查账号列表

4. **数据安全**
   - 定期备份数据库
   - 限制访问 IP（可选）
   - 启用访问日志

## 🆘 常见问题

### Q1: 忘记密码怎么办？

**解决方案：**
1. 管理员查看 Excel 密码表
2. 或使用以下命令重置：
```bash
cd /home/admin/openclaw/workspace/trust-system
python3 << 'EOF'
from app import app, db
from models import User

with app.app_context():
    user = User.query.filter_by(phone='手机号').first()
    user.set_password('新密码')
    db.session.commit()
    print('密码已重置')
EOF
```

### Q2: 域名无法访问？

**解决方案：**
1. 检查 Tunnel 是否运行：`ps aux | grep cloudflared`
2. 检查应用是否运行：`ps aux | grep "python3 app.py"`
3. 重启 Tunnel：`bash restart-tunnel.sh`
4. 检查域名 DNS 配置是否正确

### Q3: 如何添加新员工？

**方法 1：通过管理界面**
1. 管理员登录
2. 进入"用户管理"
3. 点击"添加用户"
4. 填写信息并保存

**方法 2：通过 Excel 导入**
1. 准备 Excel 文件（包含姓名、手机号、工号）
2. 管理员登录
3. 进入"数据导入"
4. 上传 Excel 文件

### Q4: 如何修改信托数据？

**当前版本：** 只读模式

**修改方法：**
1. 更新 Excel 源文件
2. 重新导入数据
3. 或直接修改数据库：
```bash
sqlite3 /home/admin/openclaw/workspace/trust-system/instance/trust.db
UPDATE trust_data SET grant_shares=1000000 WHERE employee_no='工号';
```
