## 🚀 推荐方案

### 方案 A：Cpolar 国内穿透（最简单）

**优点：**
- ✅ 国内服务器，速度快
- ✅ 中文界面
- ✅ 免费可用
- ✅ 支持 HTTPS

**缺点：**
- 免费版地址随机变化
- 付费版 ¥9/月起固定地址

**配置步骤：**

1. **注册账号**
   - 访问：https://www.cpolar.com
   - 免费注册

2. **获取 Token**
   - 登录后进入控制台
   - 复制认证 Token

3. **配置隧道**
   ```bash
   # 认证
   /tmp/cpolar authtoken YOUR_TOKEN
   
   # 启动隧道
   /tmp/cpolar http 5001
   ```

4. **获取地址**
   - 控制台显示访问地址
   - 如：`http://abc123.cpolar.cn`
